## Описание программы (кратко)
Генератор случайных паролей — десктопное приложение на Python с графическим интерфейсом (Tkinter) для создания надёжных паролей с настраиваемыми параметрами. Программа позволяет генерировать пароли заданной длины с выбором типов символов, копировать их в буфер обмена и сохранять историю генерации в JSON‑файле.

### Возможности (что программа умеет делать)
* **Генерация паролей** с настраиваемой длиной (от 4 до 32 символов).
* **Выбор типов символов** для включения в пароль:
    * цифры (0–9);
    * буквы (латинские, верхний и нижний регистр);
    * специальные символы (`!@#$%^&*()` и т. д.).
* **Валидация ввода**:
    * проверка длины пароля (диапазон 4–32 символа);
    * требование выбрать хотя бы один тип символов.
* **Отображение сгенерированного пароля** в текстовом поле интерфейса.
* **Копирование пароля в буфер обмена** одним нажатием кнопки.
* **Сохранение истории** всех сгенерированных паролей между запусками в файле `history.json`.
* **Визуальное отображение истории** в виде таблицы с прокруткой.
* **Автоматическое обновление** таблицы истории после каждой генерации.
* **Визуальная обратная связь** через всплывающие сообщения об ошибках или предупреждениях.


## Инструкция по запуску

### Требования для запуска программы
* Python 3.6 или выше.
* Стандартные библиотеки Python:
    * `tkinter` (GUI‑фреймворк);
    * `messagebox` (модуль tkinter для всплывающих окон);
    * `ttk` (расширение для tkinter);
    * `random` (генерация случайных символов);
    * `string` (доступ к наборам символов);
    * `json` (работа с JSON‑файлами);
    * `os` (проверка существования файла).

Программа не требует установки дополнительных пакетов — все зависимости входят в стандартную поставку Python.

2. Убедитесь, что у вас установлен Python 3.6+.
3. Откройте терминал или командную строку.
4. Перейдите в директорию, где сохранён файл программы.
